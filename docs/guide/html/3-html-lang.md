### 语言属性

根据 HTML5 规范：

- 强烈建议为 html 根元素指定 lang 属性，从而为文档设置正确的语言。这将有助于语音合成工具确定其所应该采用的发音，有助于翻译工具确定其翻译时所应遵守的规则等等。

```html
<html lang="en-us">
  <!-- ... -->
</html>
```

更多关于 <font color=#d44950>lang</font> 属性的知识可以从 [此规范][langrule] 中了解。
