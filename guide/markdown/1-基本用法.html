# 无序列表

前面用 - 或 * 表示

- 列项1 
- 列项2 
- 列项3
1
2
3

输出

  • 列项1
  • 列项2
  • 列项3

# 有序列表

前面用数字 + . 表示

1. 列项1
2. 列项2
3. 列项3
1
2
3

输出

  1. 列项1
  2. 列项2
  3. 列项3

插入链接与插入图片的语法很像，区别在一个 ! 号

3

# 粗体与斜体

Markdown 的粗体和斜体也非常简单，用两个 * 包含一段文本就是粗体的语法，用一个 *包含一段文本就是斜体的语法。

**这里是粗体** *这里是斜体*
1
